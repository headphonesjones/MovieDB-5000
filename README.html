Foundation libsass template

Start your own local project that uses Grunt and libsass!

Requirements

You’ll need to have the following items installed before continuing.

  • Node.js: Use the installer provided on the NodeJS website.
  • Grunt: Run [sudo] npm install -g grunt-cli
  • Bower: Run [sudo] npm install -g bower

Quickstart

bash git clone git@github.com:zurb/foundation-libsass-template.git npm install && bower install

While you’re working on your project, run:

grunt

And you’re set!

Directory Structure

  • scss/_settings.scss: Foundation configuration settings go in here
  • scss/app.scss: Application styles go here
